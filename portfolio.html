The Assist application connects users with volunteer and community engagement opportunities. This app uses JavaScript, MongoDB, Express, React, Redux, Node.js, JWT, Mongoose, MDBootstrap, and The Charity Navigator API.

The Audiophyle application allows users to search for their favorite musical artists, discover new artists, and save their selections to their profile. This app uses MySQL, JavaScript, Node.js, Express, Bootstrap, and TheAudioDB.

The Peckish application helps users search for and discover new and exciting food and drink recipes without having to purchase a cookbook. This app uses HTML, CSS, jQuery, JavaScript, UiKit, and TheMealDB and TheCocktailDB APIs.

The Burger Place is an application created using MySQL, Node, Express, Handlebars and an ORM. The MVC design pattern is utilized. Node and MySQL query and route the data in the application while Handlebars generates the HTML.

The Burger Place

The Fitness Tracker application creates and tracks daily workouts. Users can log multiple exercises in a workout on a given day. Users can track the name, type, weight, sets, reps, and duration of exercise. If the exercise is a cardio exercise, users can track distance traveled. This app uses MongoDB, Mongoose, Express, NoSQL, and JavaScript.

Fitness Tracker

The Team Profile Generator application allows a user to manage information for their team. Using the command-line, a user can input their team's information. When the user has completed building the team, the application will create an HTML file that displays a nicely formatted team roster based on the information provided by the user.
